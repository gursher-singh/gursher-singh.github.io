Spearheaded the deployment of network scanning functionality across 12 provincial park locations on Windows 11. Packet Analysis with Wireshark

Utilized Wireshark to perform live packet captures, diagnosing SMB traffic blocks. Pinpointed OS, firewall, and network-level restrictions to resolve connectivity failures in non-standard environments.

SMB Share & NTFS Permission Configuration

Configured secure SMB shares and NTFS permissions on Windows 11 endpoints. Aligned network discovery settings and access controls to ensure reliable scanner write access without compromising system security.

Windows Defender Firewall Remediation

Resolved inbound SMB blocks by creating Windows Defender Firewall rules to allow specific port traffic on the right network profiles while maintaining strict baseline security postures.

Pilot-to-Province Scalability

Documented the findings into a comprehensive, repeatable rollout strategy. Documented identified issues and troubleshooting steps to accelerate province-wide implementation.
